# 股市交易系统

基于《股市修行录》的完整A股交易体系网站。

## 🎯 项目特点

- **系统性**：从新手到高手的完整学习体系
- **实用性**：提供具体的操作模板和检查清单
- **创新性**：独特的"见山三境界"框架
- **实时性**：动态市场数据和情绪指标

## 📱 部署说明

### GitHub Pages 部署
1. 上传到 GitHub 仓库
2. 设置 Pages → Source: Deploy from a branch
3. 选择 main 分支和 / (root) 目录
4. 自动获得公网访问地址

### 本地运行
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

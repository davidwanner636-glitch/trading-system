# 📊 A 股市场情绪数据 API 使用说明

## 🎯 功能说明

情绪温度计 Pro 现在使用**真实的 A 股市场数据**，通过 AKShare 自动获取并计算五大维度情绪指数。

## 📡 数据源

- **AKShare** - 开源免费的 Python 金融数据接口库
- **东方财富** - 实时行情数据
- **更新频率** - 每 30 秒自动刷新

## 🚀 启动方法

### 第 1 步：安装依赖

```bash
pip install akshare flask flask-cors
```

```bash

python3 market_data_api.py
```

### 第 3 步：访问网站

打开浏览器访问：
```
http://localhost:5000/api/market-emotion
```

## 📊 五大维度计算逻辑

### 1. 📈 成交量维度
- **数据源**：两市成交额
- **评分标准**：
  - < 8000 亿：0-50 分（缩量）
  - 8000 亿 -1 万亿：50-80 分（平量）
  - > 1 万亿：80-100 分（放量）

### 2. ⚖️ 涨跌比维度
- **数据源**：全市场上涨/下跌家数
- **评分标准**：上涨家数占比 × 100

### 3. 👑 龙头维度
- **数据源**：涨停板连板高度
- **评分标准**：
  - 7 板+：90 分
  - 5-6 板：75 分
  - 3-4 板：60 分
  - < 3 板：40 分

### 4. 🔥 题材维度
- **数据源**：板块涨幅排行
- **评分标准**：
  - 平均涨幅>3%：85 分
  - 平均涨幅>2%：70 分
  - 平均涨幅>1%：55 分
  - < 1%：40 分

### 5. 🌍 大盘维度
- **数据源**：上证指数点位
- **评分标准**：
  - > 3300 点：85 分
  - 3100-3300 点：70 分
  - 2900-3100 点：55 分
  - < 2900 点：35 分

## 🔄 自动更新

- **前端**：每 30 秒自动调用 API 刷新数据
- **显示**：弹窗顶部显示最后更新时间
- **动画**：加载时显示旋转图标

## ⚠️ 注意事项

1. **需要 Python 环境**：确保已安装 Python 3.8+
2. **AKShare 依赖**：首次使用会自动下载数据
3. **网络要求**：需要访问东方财富等数据源
4. **交易时间**：非交易时间数据可能不更新

## 🛠️ 故障排查

**问题 1：API 启动失败**
```bash
# 检查端口是否被占用
lsof -i :5000
# 更换端口
python3 market_data_api.py --port 5001
```

**问题 2：数据获取失败**
- 检查网络连接
- 检查 AKShare 版本：`pip install --upgrade akshare`
- 查看错误日志

**问题 3：前端无法连接**
- 确保 API 服务已启动
- 检查浏览器控制台是否有 CORS 错误
- 确认 API 地址正确
